Default class namespace: `bulma-`

------

Configure your preferred class namespace in **postcss.config.js**.

#### Initialize

```
npm install
```

#### Build Bulma with all elements, components, etc.

```
npm run build
```

#### Test the web page into Github Workspce

```sh
npm i -g http-server
```

```sh
http-server
```
